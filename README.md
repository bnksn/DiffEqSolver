## Requirements
- CMake
- Make
- Any C++ compiler that supports C++23
- Python 3 (including Matplotlib and Numpy)

## How to build
1. Clone the repository
2. cd into the root directory
3. Run "cmake -B ./build"
4. Run "cmake --build ./build"
